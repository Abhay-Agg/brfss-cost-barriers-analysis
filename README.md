# Cost Barriers, Preventive Care, and Mental Health in U.S. Adults

This repository contains a reproducible public health analysis using CDC BRFSS data. The project examines how cost-related barriers to care are associated with preventive care use and frequent mental distress among U.S. adults.

## Project goals
- Build a clean, documented analytic workflow from raw public health data
- Perform data auditing, cleaning, validation, and recoding
- Conduct descriptive and statistical analysis
- Present findings in plain language for research, healthcare, and nonprofit audiences

## Planned workflow
1. Data audit
2. Cleaning and recoding
3. Validation checks
4. Exploratory analysis
5. Statistical analysis
6. Final summary report

## Repository structure
- `data/` raw, interim, and processed datasets
- `docs/` data dictionary, methodology, and limitations
- `notebooks/` step-by-step analysis notebooks
- `src/` reusable Python functions
- `outputs/` tables, charts, and report files
- `tests/` validation tests

## Data source
CDC Behavioral Risk Factor Surveillance System (BRFSS)
